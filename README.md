# 🛴 Samokat Analytics — Тестовое задание

Продуктовая аналитика для интернет-магазина «Самокат». Проект выполнен на Python с использованием pandas, matplotlib и seaborn.

---

## 📊 Ключевые результаты

| Метрика | Значение |
|---|---|
| **Самая ходовая категория** | [Безалкогольные напитки/Напитки] |
| **Средний чек за 13.01.2022** | [915.64] руб. |
| **Доля промо в категории «Сыры»** | [30,7]% |
| **Категория с максимальной маржой** | [Безалкогольные Напитки] |
| **Категория с минимальной маржой** | [Специальное питание] |
| **ABC-группа лидеров (подкатегории)** | AA: [23] шт., AB: [9] шт. |

---

## 🛠 Стек

- **Python** 3.x
- **pandas** — обработка данных
- **matplotlib**, **seaborn** — визуализация
- **numpy** — вычисления
