# Тестовое задание: Продуктовый аналитик (Самокат)

## Описание
Данный кейс предоставлялся в качестве тестового задания на позицию продуктового аналитика в компанию "Самокат".

## Данные

### Таблица `products`
| Поле         | Описание               |
|--------------|------------------------|
| `product_id` | ID товара              |
| `level1`     | Категория              |
| `level2`     | Подкатегория           |
| `name`       | Наименование товара    |

### Таблица `orders`
| Поле            | Описание                  |
|-----------------|---------------------------|
| `order_id`      | Номер чека                |
| `accepted_at`   | Дата и время чека         |
| `product_id`    | ID товара                 |
| `quantity`      | Количество товара в чеке  |
| `regular_price` | Регулярная цена           |
| `price`         | Текущая цена              |
| `cost_price`    | Закупочная цена           |

## Задания

1. **Самая ходовая товарная группа**
   - По какой категории продано больше всего позиций?
   - Таблица + barchart.

2. **Распределение продаж по подкатегориям**
   - Распределение количества проданных позиций в каждой категории по подкатегориям.
   - Расчётная таблица.

3. **Средний чек в заданную дату**
   - Какой средний чек был 13.01.2022?

4. **Доля промо в заданной категории**
   - Доля промо-продаж от общего объёма категории "Сыры" (в штуках).
   - Пайчарт.

5. **Маржа по категориям**
   - Маржа в рублях и процентах по всем категориям `level1`.
   - Два горизонтальных барчарта.

6. **ABC-анализ**
   - ABC по количеству продаж.
   - ABC по сумме продаж.
   - Итоговая группа (например: `A C`).
   - Анализ на уровне подкатегорий.
